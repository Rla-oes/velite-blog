# 대형언어모델 입문

## 1. 대형 언어 모델의 시대

- 최근 중국에서는 200개 이상의 스타트업이 자체 LLM을 개발 중
- 이는 LLM 기술이 더 이상 OpenAI, Google, Meta 같은 빅테크만의 전유물이 아님
- 실제 LLM 기술은 점차 **대중화**, **도메인 특화**, **국가 전략화**되는 추세

---

## 2. 법률 도메인에서의 LLM 적용 현황

### 현실성과 한계
- **GPT-4**는 미국 변호사 시험을 통과했고, 초기에는 **상위 10% 성적**이라 발표
- 이후 재검토 결과, 실제로는 **상위 50% 수준**, **주관식은 하위 30%대**에 불과

### 환각(Hallucination) 문제
- 없는 판례나 법령을 **그럴듯하게 생성**
- Stanford, MS, OpenAI, Meta 등에서 측정한 환각률은 **70–90% 수준**
- Gemini vs GPT-4 비교에서도, 전문 도메인에서는 **fine-tuning 없이는 성능 제한적**

### 한국 변호사시험 기준 성능
- GPT-4가 선택형 문제에서 **50점 전후**
- 형량 판단, 판결 요약, 논리적 정합성에서 아직 인간보다 미흡

---

## 3. 법률 NLP 주요 과제(Task)

### 분류
- 사건명 자동 분류 (형사, 민사, 행정 등)
- 판결문 → 법령 분류 자동화

### 요약
- 방대한 판결문을 구조화된 요약으로 축약

### 예측
- 민사: 배상 가능 여부 및 액수
- 형사: 유죄/무죄 예측, 양형 수준 판단

### 도전 요소
- LLM은 **상관관계를 인과로 오해**하는 경향
    - “피고가 여성일 때 형량이 낮다” → 잘못된 인과 추론으로 이어질 수 있음

---

## 4. 정보 추출 및 통계 분석

### 정량 정보 추출
- 판결문에서 다음 정보를 자동 추출:
  - 징역 기간
  - 벌금 액수
  - 사기 금액
  - 재범 여부

### 통계 기반 양형 분석
- 사기 금액과 형량의 **정량적 관계**
- **음주운전 양형 변화**: 사회 분위기 변화가 판결에 영향을 미친 사례
- 판결 결과의 시간적 변화 분석 → 사회 변화 감지 가능

---

## 5. 지도학습의 4요소와 딥러닝 도입 배경

### 지도학습 4요소
1. **모델**: 입력 데이터를 출력으로 변환하는 함수 구조
2. **데이터**: 입력–출력 쌍으로 구성된 학습 데이터셋
3. **손실함수(Cost Function)**: 예측과 실제 정답 간 차이 계산
4. **학습 알고리즘**: 손실 최소화를 위한 가중치 조정 (ex. 경사하강법)

### 딥러닝 필요성
- 기존 기계학습(SVM, 결정트리)은 고차원 분류·예측에서 한계
- 다층 퍼셉트론을 통한 고차원 매핑 가능 → LLM은 그 최정점

---

## 6. 임베딩과 Word2Vec 이해

### One-hot 인코딩 한계
- 차원이 커짐
- 단어 간 의미 유사도 파악 불가

### Word2Vec 도입
- **CBOW (Continuous Bag-of-Words)**  
  주변 단어들 → 중심 단어 예측  
- **Skip-gram**  
  중심 단어 → 주변 단어 예측

### 의미론적 연산
- 벡터 연산으로 의미 추론 가능  
  `king - man + woman ≈ queen`  
- 단어의 문맥 기반 분포를 벡터 공간으로 매핑 → 의미 학습
